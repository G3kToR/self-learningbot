# Самообучающийся чат-бот

Использует библиотеку BrainJS для машинного обучения.

- Умеет считать математические выражения.
- Умеет бросать монетку.
- Умеет немного "предполагать" (пример с магазином на скриншоте).
- На каждое действие у бота несколько фраз, которые он выбирает рандомно. Мне показалось так будет интереснее.
- Умеет запоминать при перезагрузки страницы.
- Всегда можно поправить словом "нет".
- Можно сбросить кэш словом "сброс".
